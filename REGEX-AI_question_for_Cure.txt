Using REGEX
From: "<strong>Cure</strong> 2 consecutive saves"
Extract: "<strong>Cure</strong> 2 saves" AND 2 AND "consecutive" if it exists
where "strong" can be any html tag
where 2 can be any number
where "saves" can be singular "save" 
where "consecutive" is optional

To extract the specific parts you need while keeping the HTML tag and "consecutive" flexible, use this pattern:
(<(\w+)>.*?<\/\2>)\s+(\d+)\s+(consecutive\s+)?(saves?)

Regex Breakdown
(<(\w+)>.*?<\/\2>): Matches any HTML tag and its content (e.g., <strong>Cure</strong>).

\s+(\d+)\s+: Captures the number (e.g., 2) surrounded by spaces.

(consecutive\s+)?: Optionally matches the word "consecutive" and a space.

(saves?): Matches "save" or "saves."

Captured Groups
If you apply this to <strong>Cure</strong> 2 consecutive saves, your groups will be:
Group 1: <strong>Cure</strong>
Group 3: 2
Group 4: consecutive (if present)
Group 5: saves
